DNS

Phonebook of the Internet

Computers can only communicate using series of numbers, so DNS was developed as a sort of “phone book” that translates the domain you enter in your browser into a computer readable IP. The Domain Name System (DNS) is like the phonebook of the Internet. We all access information online through domain names like; .com or .net. Web browsers need to interact through Internet Protocol (IP) addresses. So, DNS severs translate the domain names to IP addresses so browsers can load Internet resources.

The process of DNS resolution involves converting a hostname (such as www.example.com) into a computer-friendly IP address (such as 192.168.1.1). When a user wants to load a webpage, a translation must occur between what a user types into their web browser (example.com) and the machine-friendly address necessary to locate the example.com webpage.
