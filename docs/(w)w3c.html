W3C, HTML, and CSS

W3C

The World Wide Web Consortium (W3C) is an international organization committed to improving the web. It is made up of several hundred member organizations from a variety of related IT industries. W3C sets standards for the World Wide Web (WWW) to facilitate interoperability and cooperation among all web stakeholders. It was established in 1994 by the creator of the web, Tim Berners-Lee.

HTML

HTML stands for Hypertext Markup Language. HTML is not a programming language, meaning it doesn’t have the ability to create dynamic functionality. Instead, it makes it possible to organize and format documents, similarly to Microsoft Word. When working with HTML, we use simple code structures (tags and attributes) to mark up a website page. For example, we can create a paragraph by placing the enclosed text within a starting (p) and closing (/p) tag.

CSS

CSS (Cascading Style Sheets) allows you to create great-looking web pages. CSS is a language for specifying how documents are presented to users — how they are styled, laid out, etc. A document is usually a text file structured using a markup language such as HTML, the most common markup language. CSS can be used for very basic and advanced document formatting and styling. This formatter is a rule-based language — you define rules specifying groups of styles that should be applied to particular elements or groups of elements on your web page.
